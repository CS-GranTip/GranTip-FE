# GranTip
### 2025 여름 창의학기제 프로젝트

## 테스트 사이트 주소
- https://gran-tip-fe-test.vercel.app/

## 주요 기능
- 장학금 검색, 추천
- 로그인, 로그아웃, 회원정보 수정
- 장학금 즐겨찾기, 캘린더
